Stroke Prediction with ML

This project explores the relationship between Cerebrovascular accident (CVA) and lifestyle factors. Through a thorough exploratory data analyses, including heatmaps, bar charts, and pie charts, the relationship between these two variables are identified. Machine learning library scikit learn is used and Numpy and Pandas are used for Data processing. For Visualization, libraries such as matplotlib and seaborn are used. The prediction percentage is 94% and the output is presented in a csv file within the code.
